# SLIME MOLD · SLM

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **LIFE SCIENCE**.

SLIME MOLD (SLM) — minds without brains: Physarum polycephalum is a single giant cell, no neurons, that crawls through a maze and holds the shortest path to food (Nakagaki 2000), and, given oat flakes laid out as Tokyo's cities, grows a network rivaling the real rail map for efficiency and fault tolerance (Tero 2010). The 'algorithm' is physical — tubes that carry more protoplasmic flow thicken, starved tubes dissolve; it even habituates (a basic learning) and can pass that state to a fused partner (Dussutour). LIVE interactive: an illustrative tube-reinforcement network (click to add a food node). Two-layer honest; living researchers (Nakagaki/Tero/Adamatzky/Dussutour) cited not minted. Kin to the octopus, mycelium, cellular automata. 10 emergents.
